Парные скобки "()" имеют двоякое назначение.
1. Наряду с ключевым словом function они указывают на то, 
что мы имеем дело со специальным объектом - функцией.
Это важно, когда мы присваиваем функцию какой-то переменной.
Например:

function doSomething() {
    some code
}
let variable = doSomething();
variable();

Таким образом мы, и интерпретатор, сразу видим, что в локальной 
переменной хранится определение функции.

2. Также они служат контейнером для входных параметров - аргументов,
которые будут использоваться при выполнении функции.
Например:

function doSomething(num, str, bool) {
    some code
}
doSomething(3, 'some text', true);
